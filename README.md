Python Quiz Game
Overview
This is a Python-based quiz game inspired by KBC (Kaun Banega Crorepati). The game asks multiple-choice questions, tracks your score, and provides instant feedback on your answers.​

Features
Multiple-choice questions

Score tracking

Immediate feedback after each answer

End-of-game summary with final score​

How to Play
Run the Python file.

Answer each question by entering the corresponding option (A, B, C, D).

The game will show if your answer is correct or incorrect.

At the end, your total score will be displayed.​

Setup
Make sure Python is installed on your system.
Run the game using:

python Quiz.py

Game Instructions
Answer each question by typing the letter of your choice.

Each correct answer increases your score.

After all questions are answered, your final score is shown.
